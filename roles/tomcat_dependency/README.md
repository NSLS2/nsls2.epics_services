tomcat_dependency
=================

Install a general tomcat services
Based on role
<https://github.com/jmutai/tomcat-ansible>

Requirements
------------
OpenJDK 11
role: jdk_dependency

Role Variables
--------------

| Variable             | Type   | Description                                                                                      |
|----------------------|--------|--------------------------------------------------------------------------------------------------|
| `tomcat_dest`        | string | The installation location of the tomcat server                                                   |
